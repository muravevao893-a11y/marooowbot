# Notes

## Точки, которые надо проверить в живом Telegram

- Бот должен видеть auto-forward сообщения из канала в discussion-группе.
- Бот должен видеть обычные комментарии. Если не видит — отключи privacy mode в BotFather и/или дай права админа в discussion-группе.
- Для отправки подарков нужен корректный `gift_id`, доступный боту через `/gifts`, и достаточный баланс Stars.
- Numeric IDs канала и чата должны быть точными.

## Быстрый smoke test

1. Запусти Docker Compose.
2. Запусти бота.
3. Напиши боту `/start` с аккаунта админа.
4. Напиши `/admin`.
5. Создай ручной розыгрыш без gift_id (`-`), на 1 минуту, 1 победитель.
6. Нажми кнопку участия с тестового аккаунта, предварительно сделав `/start` в боте.
7. Дождись итога.
8. Потом проверь `/gifts` и укажи реальный `gift_id`.
